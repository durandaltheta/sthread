◆ service()

template<typename FUNCTOR >
static worker& st::service ( )
inlinestatic

Access a singleton instance of a worker thread.

Worker thread will be running template type FUNCTOR as its handler. A limitation of service worker threads is that type FUNCTOR is always default constructed (there's no place constructor arguments can be provided).

The worker is returned as a reference instead of a shared_ptr as a minor usage convenience because the underlying worker object is guaranteed to remain in existence for the runtime of the program.

Returns
singleton worker thread reference
